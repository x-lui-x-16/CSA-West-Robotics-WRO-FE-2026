# 🧊 Models
This directory contains all the 3D models used in the robot.

All files stored here correspond to parts designed for 3D printing and assembly of the robot structure. These models were created specifically for this project to ensure proper fit, modularity, and mechanical efficiency.

## 📌 Contents
- 3D printable robot parts
- Sensor and electronics mounts

## 🛠️ Purpose
The main purpose of these models is to provide a complete and modular mechanical structure for the robot, allowing easy assembly, replacement, and improvement of components during development and testing.
