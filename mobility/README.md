# 🚗 Mobility
This section shows the robot’s mobility system and steers.

The robot is designed with a separated propulsion and steering architecture to ensure smooth and precise movement throughout the WRO Future Engineers challenge.

## 📌 Contents
- Ackermann steering system
- Rear-wheel drive configuration

## ⚙️ System Overview
The robot uses a **rear-wheel drive system** powered by a Geartisan 12V DC motor, which provides consistent forward motion.

Steering is handled by an **MG996R servo motor**, which controls an Ackermann-style linkage connected to the front wheels. This setup allows for smooth turning behavior and reduces wheel slip during direction changes.

The combination of these two systems ensures stable navigation, accurate turning radius control, and reliable performance in dynamic track conditions.

## 🔍 Purpose
The mobility system is designed to replicate realistic vehicle behavior, improving stability and control while allowing the robot to execute precise maneuvers such as obstacle avoidance and parking.
