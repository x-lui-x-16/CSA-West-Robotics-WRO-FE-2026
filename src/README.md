# 💻 Source Code
This directory contains all the software developed for the robot.

The code is written for the Arduino Nano 33 IoT and manages all the robot’s autonomous behavior, including sensor processing, decision-making, and motor control.

## 📌 Features
- Real-time sensor data processing
- Color detection integration (Pixy2 camera)
- Obstacle detection using ultrasonic sensors
- Steering control using servo motor
- Motor control via L298N driver
- Autonomous navigation logic

## ⚙️ System Behavior
The robot continuously reads sensor inputs, processes environmental data, and executes movement decisions in real time to complete the WRO challenge autonomously.

## 🧠 Architecture
The software is structured to ensure modularity, making it easier to test, debug, and improve individual subsystems.
